# 初音未来主题提示词与控件风格

## 技术栈

- React + TypeScript
- Vite
- CSS Variables + CSS Modules（或等价方案）
- 可选：Framer Motion（用于微光呼吸与按钮高亮）

## 视觉提示词

- 青绿霓虹、清透、高科技
- 轻量几何纹理与电子线条
- 角色导向、活力与清晰感并存
- 层次分明、边界带微光

## 控件风格

- 字体：现代几何无衬线，字重 500-700
- 圆角：中高圆角（16px-22px）
- 阴影：带青绿光晕
- 边框：1px 霓虹描边
- 背景：青绿渐变 + 细网格纹理

## 颜色变量建议

- --miku-bg-1: #D8FFF4
- --miku-bg-2: #C8F7FF
- --miku-text: #0F172A
- --miku-accent: #39E6C1
- --miku-accent-2: #00B8D9
- --miku-user: #2CD3B6
- --miku-ai: rgba(255, 255, 255, 0.75)

## 关键组件规范

- 侧边栏：青绿渐变，角色入口高亮
- 主对话区：气泡边缘微光描边
- 输入框：透明底 + 霓虹描边
- 按钮：主色发光按钮
- 小图标：几何风格，端点圆润

## 动效建议

- 按钮与选中态：微光 1.5s 呼吸
- 消息出现：轻微向上漂移 + 透明度过渡
- 背景纹理：缓慢位移 90s 周期
