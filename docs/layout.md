# 页面布局与主题规范

## 前台布局（ChatGPT 风格结构）

- 左侧栏：会话列表、创建新对话、人设卡入口、模型选择
- 顶部栏：当前主题、当前人设、登录按钮、设置入口
- 主对话区：消息列表、加载状态、复制与重试
- 底部输入区：文本输入、附件（人设卡上传）、发送按钮
- 免责声明区：固定在输入区下方

## 首次打开流程

1. 弹出主题选择弹窗
2. 选择后进入聊天界面
3. 默认游客模式可直接对话

## 主题规范

### ChatGPT 风格主题

- 简洁、稳定、轻量
- 低对比度灰阶背景
- 侧栏与主区明暗区分

### 柔和主题（iOS26 毛玻璃风）

- 设计理念：温润如玉的透明感
- 形状语言：全站高圆角（20px-28px）
- 层次结构：毛玻璃叠层 + 轻柔阴影

配色建议（Soft-Glow Palette）

- 主色：#E0E7FF
- 辅助色：#FDF2F8 或 #F0FDF4
- 用户气泡：柔和实色（如 #8B5CF6）或半透明白
- AI 气泡：rgba(255, 255, 255, 0.6) + blur(10px)

动效细节

- AI 思考：呼吸光球
- 图标：圆润端点、轻微厚度感
- 动态背景：缓慢流动的渐变

跨端表现

- 手机：轻微缩放反馈、触摸玻璃感
- 电脑：层级阴影区分区域

### 初音未来主题

- 主色青绿 + 霓虹淡光
- 侧边栏突出角色卡入口
- 背景加入轻量几何纹理

## 后台布局（现代化简约）

- 顶部状态栏：今日数据、实时并发
- 左侧导航：数据概览 / 敏感词 / 公告 / 用户管理 / 模型配置 / 日志
- 主内容区：图表 + 表格 + 操作面板

## 移动端布局

- 侧栏折叠为抽屉
- 对话区占满屏幕
- 输入区固定底部
