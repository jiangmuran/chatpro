# 场景说明与交互流程

## 场景 1：首次打开

- 弹出主题选择（ChatGPT / 柔和 / 初音未来）
- 选择后进入聊天页面
- 默认游客可直接对话

## 场景 2：游客对话

- 直接输入消息发送
- 消息与主题保存在本地
- 可复制 AI 回复

## 场景 3：登录

- 点击登录按钮弹窗输入唯一字符串
- 验证后保存至本地
- 用户名、地区、时间注入系统 prompt

## 场景 4：人设卡

- 选择情景人设（如初音未来）
- 或选择专业人设（如程序架构师）
- 修改人设自动创建新对话

## 场景 5：上传人设卡

- 本地上传后立即可用
- 审核通过后公开给所有用户

## 场景 6：模型选择

- 普通、增强、专业三类模型
- 高级模型展示剩余对话次数

## 场景 7：编辑后重试

- 用户可编辑自己的内容重新生成

## 场景 8：后台运营

- 查看实时并发与平均响应时间
- 查看 token 消耗趋势与今日数据
- 查看请求最多词语

## 场景 9：合规留档

- 导出访客 IP / UA / 对话记录
- 用于审查与分析

## 场景 10：敏感词与公告

- 维护敏感词列表
- 发布公告提示

## 场景 11：用户管理

- 新建用户，用户 ID 可留空自动生成 xxxx-xxxx
- 配置高级模型额度
- 查看删除个人聊天记录
- 设定用户标签跳过敏感词审查

## 场景 12：访问限制

- 设置 IP 黑名单
- 设置每 IP 当日最高请求次数
