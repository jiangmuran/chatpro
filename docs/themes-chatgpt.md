# ChatGPT 风格主题提示词与控件风格

## 技术栈

- React + TypeScript
- Vite
- CSS Variables + CSS Modules（或等价方案）
- 可选：Framer Motion（仅用于轻量过渡）

## 视觉提示词

- 简洁、克制、稳定、专业
- 冷静的中性灰阶
- 强调信息层级与可读性
- 轻量、无侵入的边界

## 控件风格

- 字体：无衬线，字重 400-600
- 圆角：中等（10px-14px）
- 阴影：几乎不可见，强调分割线
- 边框：1px 中性灰
- 间距：紧凑但舒适，输入区与消息区间距明显

## 颜色变量建议

- --c-bg: #F6F6F6
- --c-panel: #FFFFFF
- --c-text: #111827
- --c-muted: #6B7280
- --c-border: #E5E7EB
- --c-user: #E5E7EB
- --c-ai: #FFFFFF

## 关键组件规范

- 侧边栏：暗灰背景，浅灰文字，悬浮轻微高亮
- 主对话区：浅灰背景，气泡简洁，AI 与用户区分弱色差
- 输入框：边框清晰，聚焦时轻微加深
- 按钮：扁平风格，低饱和主色
- 免责声明：轻灰小字，固定在输入区下方

## 动效建议

- 页面加载：轻微淡入
- 消息出现：向上位移 4-6px + 200ms 淡入
- 加载态：细线条闪烁或小点跳动
- 完全参考chatgpt的主页面
