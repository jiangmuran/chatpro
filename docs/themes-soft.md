# 柔和主题提示词与控件风格

## 技术栈

- React + TypeScript
- Vite
- CSS Variables + CSS Modules（或等价方案）
- 可选：Framer Motion（用于呼吸光球与缓慢渐变流动）

## 视觉提示词

- 温润如玉、半透明、柔光
- 马卡龙渐变背景、薄雾层叠
- 圆润、高亲和、弱对比
- 轻微动态、呼吸感、微 3D

## 控件风格

- 字体：圆润无衬线，字重 400-600
- 圆角：高圆角（20px-28px）
- 背景：毛玻璃叠层，backdrop-filter: blur(10px)
- 阴影：柔和扩散，避免硬边
- 边框：半透明白或主色低透明度

## 颜色变量建议

- --soft-bg-1: #E0E7FF
- --soft-bg-2: #FDF2F8
- --soft-bg-3: #F0FDF4
- --soft-text: #1F2937
- --soft-user: #8B5CF6
- --soft-ai: rgba(255, 255, 255, 0.6)
- --soft-border: rgba(255, 255, 255, 0.4)

## 关键组件规范

- 侧边栏：半透明白 + 渐变背景透出
- 主对话区：气泡层叠如薄玻璃片
- 输入框：圆角药丸形，内阴影轻微
- 按钮：圆角胶囊，轻微高光
- 状态指示：呼吸光球替代进度条

## 动效建议

- 背景渐变：极慢速漂移（60s-120s）
- 触摸反馈：轻微缩放 0.98
- 思考态：光球 2s 呼吸周期
